# Advanced Linux System Report 🚀

A comprehensive Bash-based monitoring tool developed to provide instant visibility into system health and performance. This script was built as the capstone project for **Week 1 of Linux Basics**.

## 🛠 Features

This script automates the collection of critical system metrics:
* **Real-time CPU Load:** Extracts current load averages using the `uptime` command.
* **Process Monitoring:** Identifies resource-heavy applications by listing the **Top 5 processes** consuming the most CPU.
* **Storage Analysis:** Summarizes disk health and partition usage in human-readable (GB/MB) formats.
* **Memory Tracking:** Provides a snapshot of RAM usage, including percentage utilized.
* **User Audit:** Displays a list of all currently logged-in users.
